XMLTag: A recursive XML wrapper class for reading and writing XML in C++
======================================================================

A simple multi-platform C++ xml parser
that wraps the use and creation of XML into a single object.
The internal parser is a single pass XML parser that is not only
fast but allows for midstream parsing of XML data.  The class
was designed to make using XML as a communication protocol very simple
removing the complexity from the handling of the XML that carries
the real data being communicated.

Intro
-----
XMLTag is a C++ class that represents all the general aspects of an XML tag:

- Name
- Value
- Attributes
- Children

User Guide
----------

Example Usages
--------------

```c++
#include "xmltag.h"
...
    XMLTag xml("toptag");
    XMLTag &ref = xml.addTag("innertag");
    ref.addTag("child1", "sammy");
    ref.addTag("child3", "frank"); // Will not be sorted : retains inserted order
    ref.addTag("child2", "tootie");
    
    // Print third child of inner tag
    std::cout << xml["innertag"][2].value() << std::endl;
    
    // Print tab formatted XML output
    std::cout << std::endl << xml.getXMLFormatted() << std::endl;
```

g++ -o xmltag_example1 xmltag_example1.cpp xmltag.cpp
